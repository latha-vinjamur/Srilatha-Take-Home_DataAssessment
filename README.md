# Srilatha-Take-Home_DataAssessment
## Overview
Project contains a SQL file which shows the extract and loading of the data along with the queries to show what analysis was done

## Structure
- sql/ → SQL queries used for analysis
- analysis/ → Excel file containing analysis 


## Approach
- Created a local ducked database
- Loaded the downloaded files to the tables.
- Sql file has the comments included to describe the data
- Queries were written against the database to perform analysis
- Excel sheet with the results and charts has been created to show the analysis
- Excel sheet also included the Findings which can be leveraged to cleanse the data%
